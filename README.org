* Installation
** From sources using quelpa and use-package
#+NAME: install-with-quelpa
#+BEGIN_SRC emacs-lisp
;; Install quelpa from elpa archive.
(use-package quelpa
  :ensure t
  :config
  ;; Install soong-mode form github when quelpa will be ready.
  (quelpa '(soong-mode
            :fetcher github
            :repo "bobrofon/soong-mode")))
#+END_SRC
